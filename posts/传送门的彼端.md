# 第一章：双向的奥秘

当我睁开眼睛, 发现自己站在一个 **巨大的广场** 中央。脚下铺满了 [[跌入Markdown世界]] 的石砖, 天空悬浮着无数 *闪烁的符号*。

"欢迎来到 **双向链接** 之域, "一个古老的声音回荡在空气中。

我向前走去, 看到一个巨大的 ~~破损的~~ 石碑, 上面刻着一段神秘代码：

```javascript
const destiny = links.connect(
    "传送门的彼端",
    "跌入Markdown世界"
);
```

"这就是答案!" 我突然醒悟, *两个文档本是一体*, 只是散落在时间的两端。

我 [[跌入Markdown世界]] 回头望去, 传送门正在慢慢消散...
